# Full Stack Database Web Application

This project is a full-stack web application that integrates a MySQL database with server-side and client-side functionality.

## Features
- CRUD operations
- Dynamic data display
- Database design (DDL/DML)

## Technologies Used
- MySQL
- HTML, CSS, JavaScript
